Service

品質管理を“教育”ではなく“組織構造”で解決。

  1. POINT 11部屋1名の個人制清掃員は常に1部屋1名で入室。責任の所在を個人単位で明確にします。
  2. POINT 2個人単位の品質追跡改善できない人から外されていき、品質の良い人だけが残れるストイックな環境。
  3. POINT 3チェッカー抜き打ちでチェッカーを割り当てることから、組織内の不安因子を早期発見し改善に努められます。
  4. POINT 4強大なマニュアル配置・動作・確認項目を細部まで標準化し、属人性を排除します。
  5. POINT 5報告写真の定型化同じ枚数・同じアングルで撮影し、比較・検証・履歴管理をしやすくします。
